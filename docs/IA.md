# Uso de Inteligencia Artificial

## Prompts utilizados

1. "Genera la estructura base de una SPA en React + Vite para una aplicación de agenda de eventos con login, CRUD y Bootstrap."
2. "Implementa un hook de autenticación con persistencia en LocalStorage y manejo seguro de JSON.parse."
3. "Crea un servicio Axios con interceptores para manejar errores 400, 401, 404, 500 y timeout."
4. "Diseña un componente reutilizable de alerta visual para mostrar errores sin usar alert()."
5. "Ayuda a documentar el proyecto en formato profesional para una evaluación de programación front-end."

## Justificación técnica

Las sugerencias de IA fueron útiles para acelerar la configuración inicial del proyecto y proponer estructuras claras para React, Axios y LocalStorage. Se aceptaron los enfoques que aportaban orden, separación de responsabilidades y compatibilidad con Vite. En los puntos críticos, se ajustó la propuesta para que respetara los límites del sprint y la arquitectura del proyecto, evitando sobreingeniería innecesaria.

## Reflexión del uso de IA

La inteligencia artificial fue una herramienta de apoyo para organizar ideas, proponer patrones de diseño y acelerar la implementación. Sin embargo, los cambios finales fueron revisados para asegurar que fueran coherentes con el alcance académico y la lógica del proyecto. La IA ayudó a acelerar el desarrollo, pero la decisión técnica final siempre fue evaluada por el estudiante para mantener calidad y estabilidad.

## Mejoras implementadas

- Estructura modular con componentes reutilizables.
- Integración de autenticación básica y persistencia de sesión.
- Manejo centralizado de errores HTTP.
- Persistencia del historial de búsquedas usando LocalStorage.
- Documentación profesional del proyecto para la evaluación.
