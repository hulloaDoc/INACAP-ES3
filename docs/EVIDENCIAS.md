# Evidencias de Desarrollo

## Estructura preparada para evidencias

### Login
- Captura del formulario de acceso.
- Evidencia visual del estado inicial del login.

### Dashboard
- Captura del panel principal con eventos.
- Evidencia del layout responsivo y componentes principales.

### CRUD
- Captura de creación, edición y eliminación de eventos.
- Evidencia del flujo completo del CRUD.

### LocalStorage
- Captura de la sesión persistida y del historial de búsquedas.
- Evidencia del almacenamiento local en navegador.

### Errores
- Captura de alertas de error para 401, 404, 500 y timeout.
- Evidencia del manejo centralizado de respuestas HTTP.

### Consola
- Capturas de la consola del navegador para ver logs y errores.
- Evidencia del comportamiento de la aplicación en desarrollo.

### Pull Request
- Preparar resumen de cambios y evidencia de implementación.
- Enlace o referencia al PR final cuando se genere.
